# cmantic
Terminal text editor mainly for C

## Key bindings

### Normal mode
- `i             - go to 'insert' mode`
- `gt            - jump to top`
- `gb            - jump to bottom`
- `g<linenumber> - jump to line`
- `h,j,k,l       - move left, down, up right`
- `space         - search`
- `H             - go to beginning of line`
- `L             - go to end of line`

### Insert mode
Just type away
